# Global Health Inequality Analysis

Exploring global health and economic disparities through data analysis in R — covering descriptive statistics, regression modeling, and U.S. Census demographics.

## Reports

- [Descriptive Analysis](01_descriptive_analysis.html) — Summary statistics, distributions, and correlations across global health indicators
- [Regression Analysis](02_regression_analysis.html) — Linear regression modeling of GNI vs. health expenditure, and poverty vs. academic performance
- [Census Demographics](03_census_demographics.html) — U.S. Census Bureau ACS demographic analysis

**Data sources:** WHO Global Health Observatory · California Schools Dataset · U.S. Census Bureau ACS
